# 🚀 IRA: QA + AI Automation Mentoring Workspace

> A practical, 12-week mentoring plan to help a junior functional QA engineer transition into a **Junior QA Automation Engineer** with practical AI-assisted testing skills.

This repository serves as the central workspace and guide for the mentoring program. 

Start with the **[QA + AI Automation Study Plan](qa-ai-automation-study-plan.md)**. It is the source of truth for the current roadmap.

## 📌 Guidance for Mentors & Future Agents

> [!IMPORTANT]
> Keep the plan beginner-friendly. Assume little or no programming background.

- **Primary Target:** Junior QA Automation Engineer with AI-assisted testing skills.
- **Practical over Theoretical:** Prefer practical QA examples over abstract programming lessons.
- **Focused Stack:** Stick to Python, pytest, Playwright, Postman, Git/GitHub, and AI assistant workflows.
- **Scope Limits:** Do not expand into heavy AI/ML theory, advanced algorithms, or broad software engineering unless explicitly requested.
- **Actionable Material:** When adding material, ensure it is actionable (e.g., weekly goals, small exercises, mentor prompts, or portfolio tasks).
- **Iterative Refinement:** Avoid replacing the whole plan unless asked. Append or refine focused sections instead.
